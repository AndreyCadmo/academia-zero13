# 🏋️‍♂️ Treine Mais - Landing Page

Landing Page de alta conversão desenvolvida como um Modelo Reutilizável (White Label) para o nicho de Academias, Studios de Pilates e Crossfit. O projeto foi construído focando em performance extrema, design moderno (Premium Dark Mode) e total otimização para dispositivos móveis, garantindo que usuários vindos de redes sociais como o Instagram tenham acesso rápido a planos, modalidades e localização.,

Por ser um template, toda a estrutura de textos, planos e links foi projetada para ser facilmente customizada para novos clientes em poucos minutos, tornando-se um ativo escalável para desenvolvimento freela.

## 🚀 Tecnologias Utilizadas

- **Next.js** (React Framework)
- **TypeScript** (Tipagem estática e segurança de código)
- **Tailwind CSS** (Estilização utilitária e responsiva)
- **Lucide React** / Emojis estruturados (Iconografia)

## ✨ Diferenciais do Projeto

- 📱 **Mobile-First Real:** Interface milimetricamente ajustada para celulares, com componentes adaptados para toque (`onTouchStart` + `onClick`).
- ⚡ **Alta Conversão Comercial:** Copy focado em musculação raiz, destaque estratégico para o Plano Casal (aumento de ticket médio) e valorização do serviço de Personal Trainer.
- 🎨 **Estética Premium:** Visual minimalista em tons escuros e acentos modernos (como amber-400), ideal para o nicho fitness de alto padrão.
- 🗺️ **Integração com Google Maps:** Mapa nativo incorporado perfeitamente no layout sem necessidade de APIs pesadas ou custos adicionais.
- 💬 **Chamada Direta para o WhatsApp:** Botão de conversão direta integrado com a recepção/gestão da academia.
- 📉 **Custo de Infraestrutura Otimizado:** Configurado para rodar na Vercel com custo mensal de hospedagem zerado (**R$ 0,00/mês**), exigindo apenas a manutenção anual do domínio.

## 🛠️ Como Executar o Projeto Localmente

1. Clone o repositório:



```bash
git clone https://github.com/AndreyCadmo/treine-mais.git
```

2. Entre na pasta do projeto:
```bash
cd treine-mais

```


3. Instale as dependências:
```bash
npm install
# ou
yarn install
# ou
pnpm install
```


4. Execute o servidor de desenvolvimento:
```bash
npm run dev

```


Abra [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) no seu navegador para ver o resultado.

## 📦 Build de Produção e Testes Mobile

Para testar o comportamento real do projeto em dispositivos móveis na mesma rede local sem as travas de WebSocket do ambiente de desenvolvimento, utilize o ambiente de produção otimizado:

```bash
# Gera o build limpo e otimizado
npm run build

# Inicializa o servidor de produção local
npm start

```

## 🌐 Deploy

O deploy deste projeto é feito de forma contínua através da **Vercel**, conectada diretamente a este repositório do GitHub. Qualquer alteração na branch principal disparará uma nova atualização automática em produção.
